Octopress

May 20th, 2012

Después de un tiempo utilizando tumblr traslado el blog a Octopress desplegado en github.

Octopress es un sistema de blogging basado en Jekyll. No es más que un generador estáticos de páginas escrito en ruby por mojombo cofundador de github y es el sistema que utiliza github:pages. Octopress añade un conjunto de utilidades que facilitan el trabajo con jekyll.

El principal motivo del cambio es poder escribir los posts de una forma más cómoda desde un editor de texto y no desde una web. A partir de ahora podré utilizar SublimeText o iA Writer. Con Octopress publicar trozos de código es mucho más sencillo. Tiene integración con pygments para el resaltado de sintaxis y algún otro plugin interesante como el de gist o el de jsFiddle. Todavía no estoy muy seguro de si voy a hacer una migración de los posts antiguos o no, todo depende de lo complicado que resulte.

FireQuery

Sep 2nd, 2010

FireQuery es un plugin para Firefox que se integra con Firebug. Permite inyectar jquery en cualquier página web, tiene muchas otras características , pero esta era justo la que estaba buscando cuando encontré el plugin.

Un uso práctico de este plugin sería extraer todos los links de megaupload de una página. Pongamos que queremos bajarnos Dos hombre y medio desde esta página http://www.vagos.es/showthread.php?t=842744. Lo único que tenemos que hacer es abrir la consola de Firebug e inyectar jquery dándole al botón “jQuerify”.
